# SQL E-commerce Sales Analysis

## Overview
This project analyzes e-commerce data using SQL to extract insights on sales performance, customer behavior, and product trends.

## Tools & Skills
- SQL (MySQL/PostgreSQL)
- JOINs, GROUP BY, HAVING
- CTEs & Window Functions

## Key Analysis
- Monthly revenue trends
- Top customers by spending
- Best-selling products
- Customer segmentation

## Key Insights
- Top 10 customers contribute a major share of revenue
- Certain product categories drive maximum sales
- Repeat customers generate higher value

## Project Structure
- `queries/` → SQL queries
- `dataset/` → Raw dataset
- `insights.md` → Business insights

## Dataset
Kaggle E-commerce Dataset
